# Crie um documento

Os documentos são **grupos de páginas** conectadas por meio de:

- uma **barra lateral**
- **navegação anterior/próxima**
- **versionamento**

## Crie seu primeiro Doc

Crie um arquivo Markdown em `docs/hello.md`:

```md title="docs/hello.md"
# Olá
