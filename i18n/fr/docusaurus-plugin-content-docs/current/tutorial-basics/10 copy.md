# Créer un document

Les documents sont des **groupes de pages** connectées à travers :

- une **barre latérale**
- une **navigation précédente/suivante**
- **versioning**

## Créez votre premier document

Créez un fichier Markdown à `docs/bonjour.md` :

```md title="docs/bonjour.md"
# Bonjour
