# Creează un document nou (Creaza un document nou)

Documentele sunt **grupuri de pagini (gruppi di pagine)** conectate prin:

- un **sidebar**
- **navigare anterioară/următoare (nvaigare anterioara/urmatoare)**
- **versionare (versionare)**
